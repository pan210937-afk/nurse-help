# 护士减负助手（通用版｜无登录｜单文件）
这是一个“纯静态网页”原型：不需要飞书/不需要登录/不需要数据库。

## 本地打开（Mac）
1) 解压后双击 `index.html`
2) 在浏览器里直接使用：搜索 → 点模块 → 一键复制

## 上线（让 iPhone 任何人都能直接打开）
最省事：用 Vercel（免费）或 GitHub Pages（免费）。
上线后 iPhone Safari 可“添加到主屏幕”，像 App 一样打开。

## 如何更新内容
你要改模板：用任意文本编辑器打开 `index.html`，搜索 `window.MODULES=` 后面的数据即可替换（后续我可以把“内容生成→自动更新”做成脚本）。
